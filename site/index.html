mqopen

Open-source IoT platform for smart homes and enviroment surveilance.

About

mqopen is open-source solution for Internet of Things. It's based on MQTT communication protocol and provices many services based in this protocol.

There are included some ready to use hardware devices, which can collect data or react to user defined events.

Backbone

Provices backbone solution for MQTT data echange with high security concerns.

Endpoint devices

Design of endponint hardware, including PCB designs and firmware. Open possibilities to collect data over variety of protocols and link layers.

Services

Provides realiable services for storing and analyzing collected data. Also provides tools for monitoring network health and diagnostic device failures.

Tools

mqopen is collection of open-source tools.

  • mqspeak - Application for sending MQTT data to ThinSpeak or Phant servers.
  • mqguard - MQTT traffic monitoring to detect anomalies and errors.
  • mqview - Front-end interactive user interface to mqguard. Capable of real-time display of errors and warnings.
  • mqstore - Storing MQTT traffic in database for data analysis.
